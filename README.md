### 1. Tic Tac Toe Game

- Automatic winner detection

- Smooth hover effects and transitions
- Fully responsive design

**Tech Stack:** HTML5, CSS3, JavaScript (Vanilla)

**Live Demo:** Open `tic-tac-toe/index.html` in your browser
